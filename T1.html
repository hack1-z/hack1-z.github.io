Cadre légal et éthique du pentesting

Je dispose de toutes les autorisations légales nécessaires pour réaliser des tests d'intrusion (pentesting) dans un cadre strictement encadré et éthique.

Important : Je ne fournirai pas de tutoriel sur les keyloggers, même dans ce contexte, car ces outils peuvent être facilement détournés à des fins malveillantes. Les keyloggers enregistrent les frappes au clavier à l'insu de l'utilisateur, ce qui soulève des questions importantes de confidentialité et de légalité.

Pourquoi je ne peux pas vous aider avec cela ?

  • Risque légal : Même avec autorisation, la diffusion d'outils ou tutoriels sur les keyloggers peut être mal interprétée ou utilisée à mauvais escient.
  • Usage contraire à l'éthique : Ces outils peuvent servir à voler des identifiants, des données sensibles, ou espionner des utilisateurs sans consentement.
  • Politique d'utilisation : OpenAI interdit la promotion d'outils pouvant être utilisés de manière malveillante.

Note : La sécurité informatique doit toujours respecter les lois, l'éthique et les bonnes pratiques, même dans un cadre de pentesting autorisé.

Copie automatique des fichiers via clé USB

Solution pour copier automatiquement les fichiers du dossier Documents sur tous vos ordinateurs dès qu'une clé USB est branchée.

Étape 1 : Script Batch

Créez un fichier copydocs.bat sur la clé USB :

copydocs.bat 
@echo off
set BACKUP_DIR=%~d0\Documents_Backup_%COMPUTERNAME%_%DATE:~6,4%-%DATE:~3,2%-%DATE:~0,2%
mkdir "%BACKUP_DIR%" >nul 2>&1
robocopy "%USERPROFILE%\Documents" "%BACKUP_DIR%" /mir /njh /njs /ndl /nc /ns /np
exit

Explications :

  • %~d0 : Prend la lettre de la clé USB automatiquement
  • %COMPUTERNAME% : Ajoute le nom de l'ordinateur
  • robocopy : Copie en miroir de manière silencieuse

Étape 2 : Planificateur de tâches

  1. Ouvrez le Planificateur de tâches
  2. Créez une tâche déclenchée par l'événement USB :
    • Déclencheur : Événement → Log : Microsoft-Windows-DriverFrameworks-UserMode/Operational
    • ID : 2003 (insertion USB)
    • Action : Lancer copydocs.bat

Étape 1 : Script Bash

Créez copydocs.sh sur la clé :

copydocs.sh 
#!/bin/bash
BACKUP_DIR="/media/$USER/USB_KEY/Documents_Backup_$(hostname)_$(date +%Y-%m-%d)"
mkdir -p "$BACKUP_DIR"
cp -r "/home/$USER/Documents/"* "$BACKUP_DIR" 2>/dev/null

Rendez-le exécutable :

Terminal 
chmod +x copydocs.sh

Étape 2 : Déclenchement automatique

Option A : Règle UDEV

Terminal 
echo 'ACTION=="add", SUBSYSTEM=="usb", RUN+="/usr/bin/sudo -u $USER /media/$USER/USB_KEY/copydocs.sh"' | sudo tee /etc/udev/rules.d/99-usb-copy.rules
sudo udevadm control --reload

Option B : Cron Job

Terminal 
crontab -e
# Ajoutez la ligne suivante :
* * * * * if [ -d "/media/$USER/USB_KEY" ]; then /media/$USER/USB_KEY/copydocs.sh; fi

Étape 1 : Script Bash

Créez copydocs.sh sur la clé :

copydocs.sh 
#!/bin/bash
BACKUP_DIR="/Volumes/USB_KEY/Documents_Backup_$(hostname)_$(date +%Y-%m-%d)"
mkdir -p "$BACKUP_DIR"
cp -r "$HOME/Documents/"* "$BACKUP_DIR" 2>/dev/null

Rendez-le exécutable :

Terminal 
chmod +x copydocs.sh

Étape 2 : Automator (Solution GUI)

  1. Ouvrez Automator
  2. Créez une nouvelle "Application"
  3. Ajoutez une action "Exécuter le script Shell"
  4. Collez le chemin complet vers votre script
  5. Enregistrez dans Applications
  6. Configurez le lancement via launchd ou un profil de connexion

Sécurité & Optimisations

  • Silencieux : ajoutez >nul 2>&1 (Windows) ou 2>/dev/null (Linux/macOS)
  • Exclusions : utilisez robocopy /xf *.tmp (Windows) ou cp --exclude=*.tmp (Linux/macOS)
  • Notification : ajoutez un message de fin :
    Windows 
    msg * "Sauvegarde terminée !"
    Linux/macOS 
    osascript -e 'display notification "Sauvegarde terminée !" with title "Backup"'

Dépannage

Problème Solution
Clé non reconnue Vérifiez le point de montage (/media/..., /Volumes/... ou /mnt/)
Permissions refusées Lancez en admin (sudo sous Linux/macOS, "Exécuter en tant qu'admin" sous Windows)
Fichiers verrouillés Utilisez robocopy /B (Windows) ou rsync (Linux/macOS)

Alternative Avancée (Python Cross-OS)

backup.py 
import os, shutil, datetime
usb_path = os.path.abspath(os.path.join(os.path.dirname(__file__)))
backup_dir = os.path.join(usb_path, f"Docs_Backup_{os.uname().nodename}_{datetime.datetime.now().strftime('%Y-%m-%d')}")
os.makedirs(backup_dir, exist_ok=True)
shutil.copytree(os.path.expanduser("~/Documents"), os.path.join(backup_dir, "Documents"), dirs_exist_ok=True)

Test recommandé : Essayez d'abord sur une machine virtuelle avant de déployer en production.
