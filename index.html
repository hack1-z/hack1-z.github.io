📚 Apprentissages : Conception réseau, routage dynamique, segmentation, sécurité périmétrique.

Analyse de logs et détection de patterns

Scripts Python pour analyser des fichiers de logs (auth.log, apache/access.log) et détecter des patterns d'attaques : brute-force, scans directory, requêtes suspectes. Création de visualisations Power BI pour le reporting.

📚 Apprentissages : Analyse de logs, regex, détection de patterns, visualisation de données.
PythonRegexPower BIDAXCSV

Scripts d'automatisation sécurité en Python

Développement de scripts Python pour : scan de ports, collecte automatique de logs, envoi d'alertes par email, vérification d'intégrité de fichiers. Utilisation de bibliothèques comme requests, schedule, hashlib.
